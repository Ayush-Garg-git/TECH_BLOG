# Tech Blog Web Application

A Java-based Tech Blog web application built to practice backend development concepts using **JSP, Servlets, JDBC**, and basic frontend technologies.

## Features
- User registration and login
- View and read public blog posts
- Like system (stored in database)
- Comment system (Facebook API â€“ temporary)
- Create and manage personal posts
- View post author and timestamp
- Edit user profile details

## Tech Stack
- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML, CSS, JavaScript
- Apache Tomcat

## Purpose
This project was created to apply backend concepts in a real-world scenario and understand how authentication, database interaction, and user flows work together.

## Future Improvements
- Environment-based database configuration
- Replace Facebook comments with custom comment system
- Pagination and search
- Spring Boot migration
